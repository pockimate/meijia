## 🎯 设计特点总结

### 排版系统
```css
/* 品牌名 */
text-sm tracking-[0.3em] uppercase font-light

/* 导航 */
text-[9px] tracking-[0.2em] uppercase font-light

/* 标题 */
text-3xl font-light tracking-widest uppercase

/* 正文 */
text-xs font-light

/* 按钮 */
text-[10px] tracking-[0.4em] uppercase

/* 角标 */
text-[7px] w-4 h-4 rounded-full
```

### 颜色系统
```css
/* 主色 */
bg-white (背景)
bg-black (按钮、角标)
text-gray-900 (主文字)
text-gray-400 (次要文字)

/* 边框 */
border-gray-100 (极浅)
border-gray-200 (浅)
```

### 动画时长
```css
/* 快速 */
duration-300 (透明度)

/* 中速 */
duration-500 (滑动、变换)

/* 慢速 */
duration-700 (图片切换)
duration-1000 (图片缩放)
```

### 间距系统
```css
/* 导航 */
h-16 (导航栏高度)
px-8 lg:px-12 (水平内边距)

/* 内容 */
pt-32 (顶部间距，避开固定导航)
py-24 (章节垂直间距)
gap-20 (大间距)

/* 网格 */
gap-x-6 gap-y-16 (产品网格)
```
